# Persona - Presentation Materials Index

## Quick Start

Para preparar la presentación:

1. **Lee primero**: `README.md` - Talking points completos y estrategia de conversación
2. **Revisa slides**: `slides_draft.md` - Contenido minimal para copy-paste a Google Slides
3. **Renderiza diagramas**: `visuals/*.mmd` → PNG usando Mermaid CLI o web editor
4. **Arma slides**: Copia contenido de `slides_draft.md` a Google Slides template de Seenka
5. **Practica**: Es una conversación, no un script - familiarízate con los talking points

## Los 3 Temas de Conversación

### 1️⃣ Contexto: "¿Qué Estamos Construyendo?"
- **Frame**: Validar entendimiento de la visión
- **Visual**: Intelligence cycle diagram
- **Key point**: No solo ads, sino investigación de algoritmos de targeting
- **Preguntas**: ¿Entendí bien? ¿Qué es más valioso?

### 2️⃣ Capacidades: "¿Cómo Lo Hacemos?" (5-6 slides)
- **Frame**: 5 pilares técnicos necesarios
- **Visuals**: Behavioral coherence journey + VNC control diagram
- **Key points**:
  - Realismo comportamental = el diferenciador
  - VNC = ventana a la vida digital
  - Inteligencia de targeting = la entrega de valor
- **Preguntas**: ¿Qué es crítico en M1? ¿Tolerancia a riesgo? ¿Nivel de visibilidad?

### 3️⃣ Ejecución: "¿Cómo Avanzamos?"
- **Frame**: Fases y puntos de decisión donde necesitas su input
- **Visual**: Timeline con checkpoints
- **Key point**: Colaborativo, no autoritario - necesitas sus decisiones
- **Preguntas**: ¿Phasing correcto? ¿Cómo coordinamos?
